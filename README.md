# Vehicle Clustering for Image Descriptors

## О проекте

Исследование выполнено для компании IntelliVision в рамках проекта Smart City/Transportation. Проверялась гипотеза о возможности использования методов кластеризации на основе нейросетевых дескрипторов для автоматической разметки изображений транспортных средств и поиска выбросов в данных.

## Данные

- 416 314 изображений транспортных средств
- 4 типа дескрипторов, полученных с разных нейросетевых архитектур:
  - EfficientNet (обучен на ImageNet)
  - OSNet (обучен для ре-идентификации объектов)
  - VDC_Color (специализирован на цветах)
  - VDC_Type (специализирован на типах кузова)

## Проделанная работа

- Предобработка дескрипторов (нормализация, PCA для снижения размерности)
- Кластеризация алгоритмом K-Means для каждого типа дескрипторов
- Оценка качества по индексам Калински-Харабаса и Дэвиса-Болдина
- Визуализация кластеров и их интерпретация
- Поиск выбросов с помощью DBSCAN (выявлено 5-8% потенциального шума)

## Результаты сравнения моделей

| Модель | Размерность после PCA | Индекс Калински-Харабаса | Индекс Дэвиса-Болдина | Интерпретация кластеров |
|--------|----------------------|--------------------------|------------------------|-------------------------|
| EfficientNet | 1807 | 2 741 | 4.69 | Не интерпретируются |
| OSNet | 235 | 11 831 | 3.06 | Отличная |
| VDC_Color | 90 | 29 558 | 2.66 | Хорошая |
| VDC_Type | 46 | 69 627 | 1.73 | Средняя |

### Анализ полученных результатов

**OSNet** показала наилучшие результаты с точки зрения интерпретации. Кластеры объединяют изображения одного автомобиля с разных ракурсов и в разном освещении. Это объясняется архитектурой модели, обучавшейся для ре-идентификации объектов. Метрики качества при этом средние, но бизнес-ценность максимальна.

**VDC_Color** продемонстрировала хорошее качество кластеризации по цвету автомобилей. Высокие метрики (индекс Калински-Харабаса 29 558) подтверждают компактность кластеров, хотя внутри одного кластера могут оказаться разные машины схожей цветовой гаммы.

**VDC_Type**, несмотря на лучшие численные метрики (69 627), показала средние результаты при визуальной интерпретации. Кластеры разделяют автомобили по типу кузова (автобусы, грузовики, легковые), но все легковые автомобили разных марок и цветов попадают в общие кластеры.

**EfficientNet** оказалась непригодной для задачи — низкие метрики и отсутствие интерпретируемости кластеров. Модель выделяет слишком общие признаки, не релевантные для транспортных средств.

**Поиск выбросов.** С помощью DBSCAN выявлено 5-8% потенциальных выбросов: изображения с необычными ракурсами, ветками деревьев, бликами. Это позволяет автоматизировать очистку датасета.

## Рекомендации

- Для группировки изображений по уникальным транспортным средствам использовать дескрипторы **OSNet** с K-Means
- Для анализа цветового распределения использовать **VDC_Color**
- Для очистки датасета от бракованных изображений применять DBSCAN к дескрипторам **OSNet** (отсечение 5-7% в качестве шума)
- Модель **EfficientNet** не рекомендована для данной задачи

## Заключение

Гипотеза подтвердилась: методы кластеризации на основе нейросетевых дескрипторов позволяют эффективно решать задачу автоматической разметки изображений транспортных средств. Наилучшие практические результаты получены при использовании модели OSNet.
